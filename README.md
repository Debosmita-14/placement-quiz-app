# 🎯 Placement Quiz Generator

An interactive quiz application designed to help students prepare for campus placements with practice questions in Aptitude, Data Structures & Algorithms, and Programming Languages.

![Placement Quiz App](https://img.shields.io/badge/React-18.2.0-blue) ![Vite](https://img.shields.io/badge/Vite-4.3.9-purple) ![License](https://img.shields.io/badge/license-MIT-green)

## ✨ Features

### 📚 **Comprehensive Question Bank**
- **140+ Practice Questions** across three major categories
- **Quantitative Aptitude** - Time & Work, Percentages, Probability, Profit & Loss, Speed & Distance
- **Data Structures & Algorithms** - Arrays, Trees, Graphs, Sorting, Dynamic Programming
- **Programming Languages** - Python, Java, C++, OOP Concepts, SQL

### 🎓 **Smart Learning System**
- ✅ **Multiple Difficulty Levels** - Easy, Medium, Hard
- ✅ **Timed Quizzes** - 10-minute countdown with auto-submission
- ✅ **Adaptive Mode** - Adjusts difficulty based on performance
- ✅ **Company-Specific Preparation** - TCS, Infosys, Amazon, Google, Microsoft, etc.
- ✅ **Instant Evaluation** - Automatic scoring with detailed explanations
- ✅ **Progress Tracking** - Monitor improvement across categories
- ✅ **Answer Review** - Detailed solutions for every question

### 🎨 **User Experience**
- 🚀 Fast and responsive interface
- 📱 Mobile-friendly design
- 🎯 Intuitive navigation
- 📊 Visual progress dashboards
- ⏱️ Real-time timer
- 🏆 Achievement tracking

## 📦 Installation & Setup

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Git

### Local Development

1. **Clone the repository**
```bash
git clone https://github.com/Debosmita-14/placement-quiz-app.git
cd placement-quiz-app
```

2. **Install dependencies**
```bash
npm install
```

3. **Start development server**
```bash
npm run dev
```

4. **Open browser**
```
Navigate to http://localhost:5173/
```

### Build for Production
```bash
npm run build
```

The optimized files will be in the `dist` folder.

### Preview Production Build
```bash
npm run preview
```

## 🎮 How to Use

### 1. **Select Category**
Choose from Aptitude, DSA, or Programming Languages

### 2. **Configure Quiz**
- Select difficulty level
- Enable adaptive mode (optional)
- Choose company-specific patterns (optional)

### 3. **Take Quiz**
- Answer 10 questions
- Navigate between questions
- Watch the timer

### 4. **Review Results**
- See your score and time taken
- Review correct and incorrect answers
- Read detailed explanations
- Track your progress over time

## 📊 Question Categories

### Quantitative Aptitude (40 Questions)
- **Easy:** Basic percentages, ratios, speed-distance problems
- **Medium:** Work-time, compound interest, boat-stream problems
- **Hard:** Permutations, combinations, complex probability

### Data Structures & Algorithms (40 Questions)
- **Easy:** Array operations, basic complexity, stack/queue
- **Medium:** Sorting algorithms, hash tables, graph basics
- **Hard:** Advanced algorithms, dynamic programming, tree balancing

### Programming Languages (40 Questions)
- **Easy:** Syntax, data types, basic OOP concepts
- **Medium:** Inheritance, decorators, method overloading
- **Hard:** Memory management, GIL, design patterns
